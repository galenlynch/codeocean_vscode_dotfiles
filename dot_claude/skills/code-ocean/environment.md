# Environment Configuration

## Dockerfile
Auto-generated by Code Ocean from `.codeocean/environment.json`. **Do NOT edit directly** - it is controlled by the Code Ocean UI. Direct edits will break the Code Ocean environment editor. To change the build environment, use `environment/postInstall` or the Code Ocean UI.

## .codeocean/environment.json
Written by the Code Ocean UI and uses **tabs** for indentation with **no trailing newline**. When resolving merge conflicts or editing manually, preserve both — don't let an editor/formatter add a final newline or convert tabs to spaces, or the CO UI will show spurious diffs on its next write.

The first line is a `# hash:sha256:...` comment. This is the SHA-256 hash of the Dockerfile contents *below* the hash line (everything after the first newline). To compute it:
```bash
tail -n +2 environment/Dockerfile | sha256sum | cut -d' ' -f1
```

## postInstall
Shell script that runs once during Docker image build, then cached. Typical pattern:
1. Install `uv` package manager
2. Install Python dependencies via `uv pip install --no-cache --system`
3. Install Code Server extensions (for Cloud Workstation IDE)

## metadata/metadata.yml
Capsule metadata: name, description, authors, affiliations.
```yaml
metadata_version: 1
name: "My Capsule"
description: "Brief description"
authors:
  - name: "Author Name"
    affiliations:
      - name: "Institution"
```
